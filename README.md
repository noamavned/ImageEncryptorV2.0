# Image Encryptor

Encrypt and conceal text within an image / retrieve hidden text.

## Technologies Used
- HTML
    - JavaScript
    - CSS
- Python
    - [Flask](https://flask.palletsprojects.com/en/3.0.x/)
    - [Tkinter](https://docs.python.org/3/library/tkinter.html)
